Updated m3u links of YouTube live channels, **auto-updated every 2 hours**.


### Usage
Paste this URL: 'https://raw.githubusercontent.com/useryahya/YouTube_to_m3u/main/youtube.m3u' to any player which supports M3U playlists

### Run the tool on your local machine
``` bash
git clone https://github.com/useryahya/YouTube_to_m3u.git
cd YouTube_to_m3u
chmod +x autorun.sh
./autorun.sh
```

Do not forget to add a cron job set for every 4 hours(or 5) if you plan to run the script locally.
